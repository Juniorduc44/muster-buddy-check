1. A task routes you to a `knowledge/<tech>/` folder (see root `CLAUDE.md`).
2. Read the cached `.md` if it exists.
3. If not, open that folder's `SOURCES.md`, fetch the official page, save a trimmed markdown copy, and log it below.
4. Prefer official docs. Record the URL + fetch date in each saved file so we know when it's stale.
